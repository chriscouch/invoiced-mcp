Search Tips

Invoiced supports writing more advanced queries than a simple text search to help you find specific data.

You can filter search results to documents that match specific criteria. Only fields that are indexed may be used as filters.

High value past due invoices: total:>1000 status:past_due

Find customers in certain cities: city:(Austin OR San Francisco)

Payments using specific card:
charge.payment_source.last4:4242 charge.payment_source.brand:visa

Search by transaction ID or check #: reference:1234

Search all metadata values: metadata:"West Division"

Equal: currency:usd
Not Equal: -currency:usd
Greater Than: balance:>0 or balance:>=1000
Less Than: balance:<100 or balance:<=500

Search for transactions that fit in a specific time range.
date:>2021-01-01
date:[2021-01-01 TO 2021-12-31]

Restrict results to a specific object type: is:customer
